# TIER 1 — EARLY ECONOMIC DIRECTION

```text id="h5q2va"

```

## 1. PMI / ISM

Most forward-looking broad activity signal.

Tracks:

* orders
* production
* hiring
* demand.

Very important for:

* USD
* yields
* equities
* commodities.

# TIER 2 — INFLATION PRESSURE

These answer:

```text id="n8v4we"
Is inflation rising or falling?
```

## 2. CPI (Consumer Price Index)

Probably the single biggest volatility event globally now.

Controls:

* Fed expectations
* rate path
* yields
* USD
* risk appetite.

Very high impact.

## 3. PPI (Producer Price Index)

Earlier inflation pipeline signal.

Sometimes predicts future CPI pressure.

Useful for:

* inflation persistence analysis.
